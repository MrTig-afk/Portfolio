About & privacy

Your data stays on this machine. The only external call is to OpenRouter for categorisation, and it only ever receives the sanitised row index, cleaned description, and amount - never account numbers, balances, or names.

About this project

I am a master's student, and this whole thing started with a simple refusal: I did not want to pay for yet another budgeting app just to see where my own money goes each month!

The easy answer was to make it automatic. Link every account, let some service pull the data, forget about it. But the more I sat with that, the clearer the trade-off got: automatic meant handing my entire financial history to a company I would simply have to trust. Hard pass!

So I built the opposite. FinanceTracker reads the CSV exports I download myself, sorts them into categories, and keeps every single figure on my own machine. The only thing that ever leaves is a cleaned, anonymised description used for categorisation, and even that is stripped of anything identifying before it goes.

It is a deliberate trade: a little less convenience for full control over my own data. It is open source, so anyone can run their own copy or read exactly what it does. And I am open to work and collaboration, so if any of this resonates, the links above will reach me!
